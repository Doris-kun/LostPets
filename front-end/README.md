# Lost Pets Front-end

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Features

- View list of lost/found pets
- Filter by breed, color, area
- Post new pet with image upload
- View detailed pet information
